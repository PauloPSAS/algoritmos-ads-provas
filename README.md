# 📘 Algoritmos e Lógica de Programação - ADS

Repositório com questões de provas anteriores da disciplina de Algoritmos e Lógica de Programação, do curso de Análise e Desenvolvimento de Sistemas.

## 🎯 Objetivo

Criei este repositório com o intuito ajudar estudantes a:

- Praticar com questões de provas anteriores
- Entender resoluções passo a passo
- Identificar erros comuns
- Se preparar melhor para avaliações

## 📂 Estrutura do repositório

```bash
provas/ 
└── ano-semestre-avaliacao/
    ├── README.md   # índice da prova
    ├── q01-*.md    # questão 1 
    ├── q02-*.md    # questão 2 
    └── ...
```
